# 💸 SpendTrax

A dark, premium personal finance tracker — built as a single HTML file. No server, no database, no login required (except your PIN).

## ✨ Features
- **PIN-protected profiles** — up to 5 separate users, each with their own data
- **Add expenses** — amount, category, date, payment method, notes
- **Dashboard** — live stats, animated totals, spending pie chart, recent transactions
- **History** — full searchable & filterable log with edit/delete
- **Budget** — set monthly limits, per-category caps, and alert thresholds
- **Reports** — monthly summary, daily bar chart, 6-month trend, category breakdown, CSV export
- **Auto-save** — all data persists in `localStorage`, works offline
- **Mobile-first** — fully responsive, works on phone, tablet, and desktop

## 🚀 Deploy on Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your GitHub repo
3. No build settings needed — Vercel auto-detects static HTML
4. Click Deploy — your app is live!

## 🛠 Local Use
Just open `index.html` in any browser. No setup needed.

## 📦 Tech Stack
- Vanilla HTML + CSS + JavaScript (zero dependencies)
- Chart.js (via CDN) for charts
- Google Fonts: Space Grotesk, Inter, JetBrains Mono
- localStorage for persistent data

## 📱 Mobile
Works as a PWA-like experience. On iOS: Safari → Share → Add to Home Screen.
